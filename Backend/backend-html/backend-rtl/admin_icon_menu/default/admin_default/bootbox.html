Alert

A simple alert dialog with a single button. this is the simplest usage of Bootbox, requiring only the text of the message you wish to show.

<button type="button" class="btn btn-primary" data-plugin="bootbox" data-message="This is the default alert!" data-type="alert" data-callback="defaultalert" data-class-name="flat-buttons">Default</button>
<button type="button" class="btn btn-primary" data-plugin="bootbox" data-message="This is an alert with a callback!" data-type="alert" data-callback="callbackoption">Callback</button>
<button type="button" class="btn btn-primary" data-plugin="bootbox" data-message="This is the small alert!" data-size="small" data-type="alert" data-callback="smallalert">Small</button>
<button type="button" class="btn btn-primary" data-plugin="bootbox" data-message="This is the large alert!" data-size="large" data-type="alert" data-callback="largealert">Large</button>
<button type="button" class="btn btn-primary" data-plugin="bootbox" data-message="This alert can be dismissed by clicking on the background!" data-backdrop="true" data-type="alert" data-callback="overlayalert">Overlay</button>
<button type="button" class="btn btn-primary bootbox-button" data-plugin="bootbox" data-type="confirm" data-message="This is the default confirm." data-callback="bootboxConfirmCallback">Confirm default</button>
<button type="button" class="btn btn-primary bootbox-button" id="bootboxCustomOption">Confirm options</button>
<button type="button" class="btn btn-primary bootbox-button" id="bootboxConfirmText">Confirm button text</button>

Confirm

A confirm dialog with a cancel and a confirm button. the callback function is passed a value of true or false, depending on which button the user pressed.

Prompt

A dialog which prompts for user input. the value entered will be null if the user cancelled or dismissed the dialog; otherwise it is passed the value of the text input. 

<button type="button" class="btn btn-primary bootbox-button" data-plugin="bootbox" data-type="prompt" data-title="This is the default prompt!" data-callback="bootboxPromptCallback">Default</button>
<button type="button" class="btn btn-primary bootbox-button" data-plugin="bootbox" data-type="prompt" data-title="This is a prompt with a number input!" data-input-type="number" data-callback="bootboxPromptCallback">Number</button>
<button type="button" class="btn btn-primary bootbox-button" id="bootboxpromptCheckbox">Checkbox</button>
<button type="button" class="btn btn-primary bootbox-button" data-plugin="bootbox" data-type="prompt" data-title="This is a prompt with a password input!" data-input-type="password" data-callback="bootboxPromptCallback">Password</button>
<button type="button" class="btn btn-primary bootbox-button" id="bootboxpromptselect">Select</button>
<button type="button" class="btn btn-primary bootbox-button" data-plugin="bootbox" data-type="prompt" data-title="This is a prompt with a textarea!" data-input-type="textarea" data-callback="bootboxPromptCallback">Textarea</button>
<button type="button" class="btn btn-primary bootbox-button" id="customDialogAsOverlay">Custom dialog as overlay</button>
<button type="button" class="btn btn-primary bootbox-button" id="customDialogInit">Custom dialog init</button>
